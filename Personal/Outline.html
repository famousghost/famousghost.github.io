Overview

Custom object highlighting system implemented in Unity HDRP using stencil buffer masking. Designed for object selection and scene interaction visualization.

Technical Description

The outline effect is implemented using a stencil buffer-based masking technique. The original object is rendered first and writes a custom reference value into the stencil buffer. A second, slightly scaled version of the same mesh is then rendered.

During the second pass, stencil comparison is used to discard fragments where the original object was previously rendered. This effectively removes the interior pixels, leaving only the outer silhouette visible as an outline.

Implementation Details

  • Custom stencil reference values written during the base pass.
  • Stencil comparison in a secondary shader pass.
  • Scaled mesh rendering to create controllable outline thickness.
  • Fully custom shader implementation compatible with HDRP.

Key Characteristics

  • No screen-space edge detection required.
  • Stable and consistent outline independent of camera angle.
  • Suitable for object selection, interaction feedback, or gameplay highlighting.
  • Packaged as a reusable HDRP-compatible plugin.
