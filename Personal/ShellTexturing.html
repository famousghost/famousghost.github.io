Overview

Real-time fur rendering using the Shell Texturing technique, implemented in Unity. This method approximates volumetric fur by layering multiple offset mesh shells.

Technical Description

The effect is achieved by rendering multiple copies of the same mesh, each slightly offset along vertex normals. Each layer ("shell") samples a density texture to simulate fur strands and depth.

The technique is widely used in real-time applications to approximate fur, grass, or soft volumetric surfaces without requiring true volumetric simulation.

Implementation Details

  • Multiple shell layers generated via vertex normal extrusion.
  • Alpha-based density control to simulate fur thickness and falloff.
  • GPU Instancing used to reduce draw calls and improve rendering performance.
  • Custom shader controlling strand fading, lighting response, and shell depth blending.

Optimization Strategy

GPU Instancing allows identical meshes using the same material to be submitted in a single draw call, significantly reducing CPU overhead. This is particularly important when rendering many shell layers per object.
