Role: Gameplay/Shader developer

Technologies: C#, Unity Built-in, Shaders

Responsibilities

  • Developed a custom post-processing effect combining dithering with G-Buffer normal and depth-based outline rendering.
  • Implemented a ray-marched “ketchup” effect simulating dynamic interaction with fries.
  • Contributed to core gameplay mechanics and interaction systems.

Technical Implementation

  • Screen-space outline shader using depth and normal buffers.
  • Ray marching implementation optimized for real-time performance.
  • Careful control of step count and distance field evaluation to balance quality and performance.

What I Learned

Improved understanding of screen-space edge detection techniques and optimization strategies for ray marching to reduce performance cost while maintaining visual quality.
